Descripción del proyecto:
El proyecto consiste en el desarrollo de un sistema digital de gestión de inventario y control de 
ventas para el MiniSuper JJ, cuyo propósito es modernizar y automatizar los procesos 
administrativos del negocio que actualmente se realizan de forma manual mediante libretas y 
registros físicos. 
El sistema funcionará mediante una aplicación informática conectada a una base de datos, 
donde se almacenará toda la información del negocio de manera organizada y segura. Esta 
plataforma permitirá centralizar la información y facilitar su consulta en cualquier momento, 
evitando la pérdida de datos y reduciendo errores humanos. 
A través de una interfaz sencilla e intuitiva, el personal del negocio podrá registrar, consultar y 
actualizar la información relacionada con productos, ventas, compras, clientes y flujo de caja. El 
usuario podrá ingresar nuevos productos al inventario indicando datos como nombre, categoría, 
precio de compra, precio de venta, proveedor y cantidad disponible.

Tecnologías utilizadas:
-Whatsapp: Para comunicarnos con la dueña de el negocio de 'Mini Super JJ'.
-GitHub: Para repartir el trabajo del proyecto y organizarnos mejor.
